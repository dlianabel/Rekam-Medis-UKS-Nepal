# REKAM MEDIS UKS NEPAL

Sebuah aplikasi untuk mencatat data pemeriksaan pasien dalam lingkup Sekolah Menengah Kejuruan Negeri 1 Pemalang (Nepal). Proyek ini digarap ketika saya masih melaksanakan kegiatan praktik kerja lapangan di SMANDA, pada sekitar bulan November ke belakang tahun 2024. Menggunakan VB6 sebagai aplikasi pembuat aplikasi desktop yang diperkenalkan pembimbing PKL pada saat itu.
